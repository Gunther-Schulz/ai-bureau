## Locked anchor (current VISION)

Header tagline (working title, per HANDOFF):
> A framework for AI partnership that cultivates expert craft — through deeply intertwined co-work, productive sparring, and defensibly authored output.

Tree analogy (locked in VISION):
- **Branches** (foreground; what practitioner experiences): cultivated expert craft — growing capacity, compounding work, deepening practice
- **Trunk** (load-bearing; structural commitments): three axes — intertwining, sparring, authorship preservation
- **Roots** (anchoring): Ming's IEP research; regulatory pressure; adjacent thinkers (Schön, Kahneman, Dreyfus)

## Tree analogy as marketing scaffold

The tree could organize marketing materials at multiple altitudes:

- **Top of funnel / billboard / one-liner** → branches-led: "AI partnership that cultivates expert craft." Concrete imagery (cultivation; growth; lush canopy).
- **Mid funnel / explainer page** → branches + trunk: "Cultivated expert craft, anchored in three structural commitments — intertwining, sparring, authorship preservation."
- **Bottom funnel / technical buyer / due diligence** → all three layers: branches (outcomes) + trunk (axes) + roots (research foundations / regulatory grounding).
- **Architect / contributor / researcher audience** → trunk + roots dominant: the framework's intellectual foundations + structural integrity.

Each audience meets the tree at a different altitude. Marketing scaffold mirrors the metaphor.

## Extension candidates

If marketing surfaces a need for finer-grained framing beyond the locked branches/trunk/roots, candidates exist:

- **Fruits** — signed work-products / deliverables (Begründungen, briefs, manuscripts; what gets harvested + shipped). Useful for client-facing positioning ("the work this tree produces").
- **Seeds** — bausteine / specialists / patterns that propagate (the framework's distributability mechanism). Useful for framework-as-method positioning ("what propagates beyond a single deployment").
- **Soil** — workspace shape / deployment context (practitioner-shape vs autonomous-business-shape vs personal-OS-shape vs etc.). Useful for multi-shape positioning ("what soil this tree grows in").
- **Leaves** — individual practitioner work moments. Probably too granular; unlikely to earn place.
- **Bark** — UX / integration-boundary layer (frontend / category-collapse interface concerns). Useful for frontend-positioning conversations.

Discipline (per memory feedback): don't force any of these unless real need emerges.

## Counter-positioning vs AI-product landscape

Marketing claims should differentiate against the typical AI-product positioning:

| Default AI products | PBS positioning |
|---|---|
| Faster output | Better practitioner |
| AI does the work | AI co-works on the work |
| Tool / assistant / copilot | Co-worker / colleague / partner |
| Productivity | Cultivation + capacity |
| Replace human judgment | Sharpen human judgment |
| Save you time | Make you better at it |
| AI agent (replaces human) | AI partner (works alongside) |

Per Ming's IEP foundation: most AI products risk dulling the practitioner. PBS positions as the AI that doesn't dull — that compounds expertise rather than erodes it. The COUNTER-POSITION is genuinely structurally different, not just better at the same game.
